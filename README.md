# 小小圖書管理員：分類大挑戰

一個給小學生玩的互動式網頁遊戲。  
玩家要在倒數時間內，把左側書本拖曳到右側正確的分類櫃，挑戰分類速度與準確度。

## 線上展示（GitHub Pages）

- [https://logandev-lkc.github.io/yhbookgame/](https://logandev-lkc.github.io/yhbookgame/)

## PWA 安裝與離線使用

- 本專案已支援 PWA，可安裝到平板主畫面使用。
- 首次需有網路開啟一次，之後會快取核心檔案。
- 在網路不穩或離線情境下，重新開啟仍可進入遊戲。
- 若你更新了程式，重新整理一次可拿到最新版快取。

### 平板安裝方式

- iPad（Safari）：點分享按鈕 -> 加入主畫面
- Android（Chrome）：右上選單 -> 安裝應用程式 / 加到主畫面

## 遊戲特色

- 可自訂遊戲秒數（管理員設定）
- 拖曳互動（支援滑鼠與觸控）
- 可自訂分數與連擊加成系統（國小組 / 國中組分開）
- 難度會隨分數提高
- 手機 / 平板 / 桌機 RWD 版面
- 可自訂題庫內容（國小組 / 國中組分開）

## 玩法說明

1. 點擊「開始遊戲」進入挑戰。
2. 把左側書本拖到右側對應分類櫃。
3. 分類正確可得分，連續答對有額外加分。
4. 倒數結束後顯示結算成績與歷史最高分。

## 管理員設定（隱藏入口）

- 在開始畫面連續點擊標題「小小圖書管理員」4 次，可開啟管理員設定面板。
- 可調整：
  - 遊戲秒數
  - 國小組 / 國中組的答對分數、連擊加分、答錯扣分
  - 國小組 / 國中組題庫
- 設定會儲存在瀏覽器 `localStorage`，重新整理後仍保留。
- 可在面板中「恢復預設」。

## 題庫格式

在管理員設定中可直接貼上以下格式：

- 國小組：每行一題，格式 `書名|分類代號`
- 國中組：每行一題，格式 `書名|分類代號|難度`
- 難度請用：`basic` 或 `advanced`
- 分類代號：
  - `300` 科學類
  - `800` 文學類
  - `900` 藝術類
  - `600` 史地類

## 題庫產生提示詞（給 ChatGPT）

以下提示詞可直接複製給 ChatGPT，產生可直接貼進管理員設定的題庫內容：

```text
你是一位教材編輯，請幫我產生「圖書分類遊戲」題庫，輸出必須能直接貼上系統，不要加任何說明文字。

【輸出規格】
1) 先輸出「國小組」40 題，每行格式固定為：書名|分類代號
2) 接著輸出一行分隔文字：---JUNIOR---
3) 再輸出「國中組」40 題，每行格式固定為：書名|分類代號|難度
4) 國中組難度只能用 basic 或 advanced
5) 分類代號只能使用：
   - 300（科學類）
   - 800（文學類）
   - 900（藝術類）
   - 600（史地類）
6) 每組四個分類都要平均（每個分類 10 題）
7) 書名要自然、像真實書名，不可重複，不要使用編號或前綴符號
8) 僅輸出題庫內容，不要輸出 JSON、Markdown、解釋、標題

請直接開始輸出。
```

## 難度機制

- 0 ~ 49 分：書本會顯示分類提示色與分類標籤。
- 50 ~ 100 分：書本改為統一顏色，提示減少。
- 100 分以上：只顯示書名，挑戰更高。

## 技術說明

- 純前端：`HTML + CSS + JavaScript`
- 繪圖：`Canvas 2D API`
- 無外部套件，單檔即可執行

## 本機執行

在專案資料夾執行任一方式：

- 直接用瀏覽器開啟 `index.html`
- 或使用簡易伺服器（建議）

```bash
python3 -m http.server 8000
```

再打開：

- [http://localhost:8000/index.html](http://localhost:8000/index.html)
